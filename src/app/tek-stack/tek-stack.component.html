Technology Stack used to build this simple application

1. Server Side Programming Language : Java

2. Server Side Programming Language Framework: Spring (Spring boot)

3. Server : Embedded Tomcat server (Spring boot)

4. Database: Embedded H2 Database

5. UI / Browser Side Programming Langulage : Javascript / Typescript

6. HTML & CSS

7. Javascript Framework : Angular

8. Charing and Visualizations : D3.js

9. Build Tool: Maven

10. Deployed On: AWS Cloud
